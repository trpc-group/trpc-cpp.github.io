Detailed Description

Read-write lock based on asynchronouse semaphore, which write lock takes all available units.

Note
It can not be used cross threads (not thread-safe).

Future trpc::RWLock::ReadLock ( )
inline

Acquire a read lock.

Returns
Future<> Ready: lock object is free or only locked by readers. Not-ready: lock object is locked by a writer. Failed: semaphore is broken, or current worker thread is not the same as the one that lock object was created on.
