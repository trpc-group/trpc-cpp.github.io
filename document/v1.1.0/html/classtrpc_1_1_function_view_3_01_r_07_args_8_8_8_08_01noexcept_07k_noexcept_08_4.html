Detailed Description

template<class R, class... Args, bool kNoexcept>
class trpc::FunctionView< R(Args...) noexcept(kNoexcept)>

Deduction of noexcept is not part of C++17, actually.

See also
It's an extension: https://gcc.gnu.org/ml/gcc-patches/2016-11/msg00665.html
Note
Deduction guides are not provided. As the template parameter is always required when declaring function parameter's type.
