Detailed Description

Bytes view of NoncontiguousBuffer.

Implement the basic operation of reading each byte through an iterator on a NoncontiguousBuffer.

Note
Currently it is only used within the framework and should not be used by users. During iteration, the NoncontiguousBuffer should not be changed.
