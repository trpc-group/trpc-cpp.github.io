Detailed Description

Compression interfaces for user programing.

Typedef Documentation

◆ CompressType

using trpc::compressor::CompressType = typedef uint8_t

There are multi methods to compress data.

Compression type is used to distinguish different compression algorithms. It is expressed as integer in the range of 0 - 255 ( 0 - 127 is are planed for use by the framework).
