Detailed Description

template<typename T>
class trpc::LockFreeQueue< T >

Implementation of a thread-safe lock-free queue that supports multiple producers and multiple consumers (multiple threads simultaneously fetching/putting data).

See also
http://www.1024cores.net/home/lock-free-algorithms/queues/bounded-mpmc-queue
paper: https://www.cs.rochester.edu/u/scott/papers/1996_PODC_queues.pdf
