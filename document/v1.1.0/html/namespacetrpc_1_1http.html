◆ HeaderFieldMap

using trpc::http::HeaderFieldMap = typedef detail::FieldMap<detail::CaseInsensitiveLess>

A map represents the key-value pairs in an HTTP header or HTTP trailer.

The key is case-insensitive.

A field name labels the corresponding field value as having the semantic defined by that name. Field name is case-insensitive.

A field value dose not include leading or tailing whitespace. Field values are usually constrained to the range of US-ASCII.

RFC : https://www.rfc-editor.org/rfc/rfc9110.html#name-fields

◆ QueryParameters

using trpc::http::QueryParameters = typedef detail::FieldMap<detail::CaseSensitiveLess>

A map represents the key-value pairs in URI-Query string.

The key is case-sensitive.

The query component is indicated by the first question mark ("?") character and terminated by a number sign ("#") character or by the end of the URI.

RFC : https://www.rfc-editor.org/rfc/rfc3986#section-3.4
