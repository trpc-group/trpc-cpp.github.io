Detailed Description

Represents the header of HTTP protocol message.

It contains:

  • Request line: request method, request URI, version.
  • Status line: version, status, reason phrase.
  • Header fields: key-value pairs (Key is case-insensitive).

About status: we usually use "status" only. It also contains reason phrase if user wants to set self-defined status and reason phrase. e.g. "600 user-defined-text"
