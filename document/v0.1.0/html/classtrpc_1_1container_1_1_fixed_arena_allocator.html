Detailed Description

template<typename T, std::size_t kCacheItemSize>
class trpc::container::FixedArenaAllocator< T, kCacheItemSize >

Pre-allocated fixed-size memory pool, use std::allocator if size exceed kCacheItemSize.

Note
Used for containers which use link to store data, such as std::list/std::unordered_map/ std::mulitmap, to avoid small memory fragments.
