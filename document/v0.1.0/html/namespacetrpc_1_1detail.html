std::mt19937_64 & trpc::detail::Engine ( )
inline

Thread local random engine.

Note
std::mt19937_64 performs equally well as std::mt19937 on x86_64. So we don't bother using std::mt19937 for smaller types.
We export this method into gdt::random (instead of keeping it in some private namespace) so that people who randomly need an engine could directly use this one, rather than declaring their own.
