Detailed Description

template<class T>
class flatbuffers::trpc::Message< T >

Message is a typed wrapper around a buffer that manages the underlying slice and also provides flatbuffers-specific helpers such as Verify and GetRoot.

Since it is backed by a slice, the underlying buffer is ref-counted and ownership is be managed automatically.
