Detailed Description

template<class T>
class trpc::object_pool::SharedPtr< T >

SharedPtr is an intrusive shared smart pointer that support polymorphism.

Usage: struct A : public object_pool::SharedPtrCountBase { int a; std::string s; };

template <> struct ObjectPoolTraits { static constexpr auto kType = ObjectPoolType::kGlobal; };

trpc::object_pool::SharedPtr<A> ptr1 = trpc::object_pool::MakeShared<A>();

For more usage, see object_pool_ptr_test.cc
