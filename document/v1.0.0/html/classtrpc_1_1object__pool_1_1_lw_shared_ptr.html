Detailed Description

template<class T>
class trpc::object_pool::LwSharedPtr< T >

LwSharedPtr is an intrusive shared smart pointer that does not support polymorphism.

Usage: struct A : public object_pool::LwSharedPtrCountBase { int a; std::string s; };

template <> struct ObjectPoolTraits { static constexpr auto kType = ObjectPoolType::kGlobal; };

trpc::object_pool::LwSharedPtr<A> ptr1 = trpc::object_pool::MakeLwShared<A>();

For more usage, see object_pool_ptr_test.cc
